# Main Colors

| Name           | Hex     | Usage                     |
| -------------- | ------- | ------------------------- |
| Primary Blue   | #2563EB | Main button & active menu |
| Secondary Dark | #0F172A | Sidebar                   |
| Background     | #F8FAFC | Main background           |
| White          | #FFFFFF | Card background           |
| Success Green  | #22C55E | Success status            |
| Danger Red     | #EF4444 | Error status              |
| Warning Yellow | #F59E0B | Warning status            |

---
