# RULES.md

## Project Rules

- Gunakan Laravel 11

- Gunakan Controller Resource

- Gunakan Form Request Validation

- Gunakan API Resource JSON

- Gunakan Repository Pattern jika diperlukan

- Semua response API menggunakan format:
  {
  success,
  message,
  data
  }

- Gunakan camelCase pada variable JavaScript

- Gunakan snake_case pada database

- Jangan membuat field di luar dokumentasi

- Semua route API menggunakan middleware auth:sanctum
